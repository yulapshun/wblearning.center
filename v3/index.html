A. For I/GCSE, GCSE/GCE/IAL/IB Regular Courses (Individual/Group)
  1. Individuals/group students must notify the Centres of leave by oral or written notice two working days prior to the commencement of class. Successful application of leave can be handled with one of the following two options:
    1. The class can be re-arranged to another date within that month agreed by both the tutor and the student(s).
    2. If re-arrangement of class is not applicable, the fees of this session will be carried forward to the next month. This arrangement will be limited to once a month.
  2. If students in the same group are not all on leave, point (a)(i) is not applicable for the absentee(s). Only point (a)(ii) will be applicable.
B. For Co-organized Courses

Should a student need to be absent from a class, the tuition fee for that session will not be refund. No re-arrangement of class will be made.

If classes are cancelled due to bad weather, they will be re-arranged or postponed.
